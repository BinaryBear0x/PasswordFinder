# PasswordFinder

PasswordFinder, Windows üzerinde kayıtlı Wi-Fi profillerini listeleyip seçilen profilin şifresini `netsh` çıktısından okuyan basit bir masaüstü uygulamasıdır.

## Ne Yapar

- kayıtlı kablosuz ağ profillerini listeler
- seçilen profil için `key=clear` çıktısını işler
- şifre bilgisini arayüzde gösterir

## Teknolojiler

- C#
- Windows Forms
- `netsh`

## Notlar

- Yalnızca yerel sistemde kayıtlı profiller için çalışır
